# React: A Guide to the Fundamentals

React — это популярная JavaScript-библиотека для создания пользовательских интерфейсов. В этом посте мы разберём её основные принципы и покажем, как начать с нуля.

## Что такое React?

React позволяет создавать интерфейсы из **компонентов** — переиспользуемых строительных блоков. Вместо того чтобы управлять DOM вручную, вы описываете, как должен выглядеть интерфейс, — и React делает всё остальное.

## Основы React

- **Компоненты**
- **JSX**
- **Props** и **State**
- **Хуки** (`useState`, `useEffect`)
- Рендеринг списков и обработка событий

## Пример простого компонента

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      Вы нажали {count} раз
    </button>
  );
}
