## Next.js: Основы

Next.js - React-фреймворк для быстрых и SEO-оптимизированных сайтов.

### Ключевые моменты:

*   **SSR/SSG:** Рендеринг на сервере или статическая генерация для скорости и SEO.
*   **Файловый роутинг:** Страницы создаются в папке `pages` (например, `pages/about.js` -> `/about`).
*   **Автоматическая оптимизация:**  Удаление неиспользуемого кода и разделение на чанки.
*   **API Routes:** Создание API прямо в проекте (папка `pages/api`).

### Важные концепции:

*   **`pages`:** Где лежат файлы страниц.
*   **Components:** React компоненты для UI.
*   **`_app.js`:** Глобальные стили и Layout.
*   **`getStaticProps`:** Данные при сборке (для контента, который редко меняется).
*   **`getServerSideProps`:** Данные на каждый запрос (для динамического контента).
